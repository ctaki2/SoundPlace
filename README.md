#  SoundPlace  
### Location-Based Music Discovery Platform  

---

##  Overview  
SoundPlace is a music streaming and discovery application grounded in **physical location rather than algorithms**. Instead of relying on curated feeds, users explore music through an interactive map that connects each track to a specific neighborhood in Chicago.  

This project emphasizes:  
- Intentional listening  
- Community-driven discovery  
- Storytelling through place  

---

##  User Research  
To tailor SoundPlace to real user needs, we conducted **4 user studies with over 48 participants**. These studies focused on understanding how people discover music, interact with location-based interfaces, and engage with community-driven platforms.
