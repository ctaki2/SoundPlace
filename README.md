#SoundPlace 
Location based music discovery platform 

Overview 
Soundplace is a music streaming and discovery application grounded in physical location rather than algorithms. Music is discovered through an interactive map that ties each song to a neighborhood in Chicago.
We designed this project to emphasize intentional listening, community-driven discovery, and storytelling that allow users to explore music more meaningfully. 

User Research
To tailor SoundPlace to user needs, we conducted 4 user studies with over 48 participants total.
